# Visual Upgrade Shortlist

Generated: March 5, 2026

## Current Game State (Quick Assessment)

- Mechanically strong for a vertical slice: character creation, progression, combat, enemy AI sight/chase behavior, loot, pause/save/load, and a floor boss.
- Biggest visual weakness is content density, not rendering tech.
- Highest impact next step is more art variety: biome tiles, set dressing props, enemy/NPC variety, and combat/UI polish.

## Asset Packs To Consider (High ROI, Low Integration Risk)

1. **0x72 DungeonTileset II - Sewers** ($2)  
   Adds a new biome with matching 0x72 style and extra animated characters.  
   Link: https://0x72.itch.io/16x16-dungeontileset-ii-sewers

2. **Dungeon Tileset II Extended** (Free, CC0)  
   Adds practical dungeon pieces (stairs, torches, keys/doors, props).  
   Link: https://nijikokun.itch.io/dungeontileset-ii-extended

3. **Superdark 16x16 Fantasy RPG Characters** (Free)  
   Strong NPC variety for towns/hubs/events.  
   Link: https://superdark.itch.io/16x16-free-npc-pack

4. **Superdark Enchanted Forest Characters** (Free)  
   Extra creature/enemy variety usable for deeper floors and encounters.  
   Link: https://superdark.itch.io/enchanted-forest-characters

5. **DungeonUI + DungeonFont** (Free, CC0)

   Links:

   Links:  
   - https://stealthix.itch.io/animated-effects
